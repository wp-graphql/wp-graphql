# `init_graphql_request`

Action – intentionally with no context – to indicate a GraphQL Request has started. This is a great place for plugins to hook in and modify things that should only occur in the context of a GraphQL Request. The base class hooks into this action to kick off the schema creation, so types are not set up until this action has run!
